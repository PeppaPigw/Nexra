# Nexra Roadmap

Nexra is an active fork of Gauntlet. This file tracks short-term milestones and
keeps expectations explicit during Phase 1.

## Phase 1 (macOS-only): next 1–2 weeks

- CI: Keep PR smoke builds green on `macos-latest` and `ubuntu-22.04`.
- Docs: Improve quickstart + add a lightweight contributing guide.
- macOS dev loop: Make local build/run iterations reliable and documented.
- P0 sample: Clipboard History MVP (text-only) as a bundled plugin (view + watcher).

## Later

- Expand automated checks (format/lint/tests) as they become stable.
- Revisit broader OS support once Phase 1 goals are met.
